# Paste architecture

It is a serverless web app built on Cloudflare infrastructure using Cloudflare Workers, KV, and Turso SQLite database.

It's made with React Router 7 (Fullstack) and shadcn/ui component library with Tailwind CSS for styling.
We are using Drizzle ORM for database interactions and Zod for input validation.
Shiki is used for syntax highlighting of code snippets.
This app is deployed on Cloudflare Workers platform.
